Bachelor Project Title: 
	- Artificial Intelligence based Solution for Mental Health Issues 
	- Rules-based chatbot using sentiment analysis

Description of the problem to be solved:
	- The most complex scenario is the one for dealing with a negative emotion (such as anxiety, depression and so on). Depending on the input of the user, the bot can adapt the scenario using some type of exercises, suggesting some online guided meditations or even professional help from a doctor. More than that, the bot should have different methods to congratulate the user when he/she improves her mood.
